# Library Management System

## Overview
A Java Swing based Library Management System developed using Object-Oriented Programming concepts.

## Features
- Borrow books
- Return books
- Automatic due date calculation
- Fine calculation for late returns
- User-friendly GUI using Java Swing

## Technologies Used
- Java
- Java Swing
- OOP Concepts
- Date and Time API

## OOP Concepts Applied
- Class and Object
- Encapsulation
- Abstraction
- Event-Driven Programming
